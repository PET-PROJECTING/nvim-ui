# Change Log

All notable changes to the "Neovim dynamic cursor color" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.1.0] - 2025-02-23

### Added

- Initial release
- Dynamic cursor and status bar colors based on Neovim mode (normal, insert, visual, command, replace)
- Configurable colors per mode via `nvim-ui-modes.colorCustomizations` setting
